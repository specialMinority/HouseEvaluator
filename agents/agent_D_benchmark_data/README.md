ワーキングホリデー日本人向け 1R/1K 月額家賃ベンチマーク（生データ）
調査目的

ワーキングホリデー等で首都圏に滞在する韓国人が現地家賃感覚を掴めるよう、**東京23区、横浜市の区単位、さいたま市の区単位および東京通勤圏の主要都市（川口・蕨・戸田・越谷・市川・船橋・浦安・千葉市区等）**を対象にワンルーム（1R）と1Kの平均賃料を収集しました。所得税や管理費等を考慮した再計算ではなく、各ページに記載された数字をそのまま転記しています。データは後続の集計やメディアン算出用にそのまま使えます。

データ収集方法

対象地域の選定 – ユーザーから指定された東京23区、横浜市の各区、さいたま市の各区、および埼玉・千葉の東京通勤圏主要市区を一覧にし、1R/1Kの平均賃料を調査することとした。

CHINTAIでの検索 – まずCHINTAIの各地域ページを開き、1Kの家賃相場表と1Rの家賃相場表を参照。東京都や横浜市、さいたま市では区別の表があり、各区の平均賃料を記録した。ページ冒頭の説明から集計条件と更新日を確認。

LIFULL HOME’Sでの検索 – CHINTAIに相場表がない市区（川口市・蕨市・戸田市・越谷市・市川市・船橋市・浦安市・千葉市各区）についてはLIFULL HOME’Sの家賃相場ページを開き、ワンルームと1Kの平均賃料を記録した。LIFULLの説明文から算出方法と更新頻度を確認。

数値の変換 – 各ページの平均家賃は“○○万 円”表記で示されているため、数値を万円から円単位に変換（例：9.05 万→90,500 円）した。管理費を含まない物件価格であることを注記した。

データ構造化 – 収集した情報をCSVとJSONの2形式でまとめるため、region_country（国コード）、prefecture（都道府県）、municipality（市区名）、layout_type（1R/1K/1DK/1LDK）、building_structure（建物構造: wood/light_steel/steel/rc/src/other/all）、avg_rent_yen（円単位の平均賃料）、source_name、source_url、source_updated_at、collected_at（収集日）、method_notesを含む行/オブジェクトとして整理した。source_updated_atはCHINTAIページで明記されている更新日（2026‑02‑18など）を使用し、LIFULLは特定の日付が記載されていないためnullとした。なお、既存の構造未区分データは building_structure=all として扱い、構造別データがある場合はそれを優先的に利用する想定。

注意点

推定や補完は行っていない – 物件数が少なく1Rまたは1Kの相場が表示されていない自治体（例：横浜市泉区の1R）については行を作成していない。数値の推定や欠損補完はしていないため、利用時には注意が必要。

更新日と基準の違い – CHINTAIは2026年2月18日更新のデータであり、築20年以内の掲載物件の平均賃料を基準にしている。LIFULL HOME’Sは毎週金曜日に更新され、駅徒歩10分以内の賃貸物件データを過去データから算出している。基準が異なるため、異なる出典同士の単純比較には注意すること。

管理費・共益費の扱い – 両サイトとも管理費や駐車場代を除いた賃料のみで平均を算出している。実際の居住費用には別途管理費等が加算される可能性がある。
